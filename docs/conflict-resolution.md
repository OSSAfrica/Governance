# Conflict Resolution

This document provides a clear process for resolving governance and
collaboration conflicts.

## Principles

- Fairness
- Timeliness
- Confidentiality where needed
- Respectful communication

## Resolution Path

1. Direct discussion between involved parties.
2. Mediated review by a Core Team member.
3. Escalation to OSSAfrica SIG for unresolved cases.
4. Final escalation to OpenSSF BEAR WG for governance-level disputes.

## Documentation

Maintain a brief record of:

- Parties involved
- Timeline
- Actions taken
- Final outcome
