# Working Groups

This document outlines how working groups are formed and operated in
OSSAfrica.

## Purpose

Working groups execute focused initiatives that support OSSAfrica strategy,
operations, and community growth.

## Formation Criteria

A new working group should define:

- Problem statement and scope
- Initial leads and member roles
- Expected deliverables
- Reporting cadence
- Success metrics

## Governance Expectations

- Operate transparently with public notes when possible.
- Follow OSSAfrica Code of Conduct and governance policies.
- Keep decisions documented and discoverable.

## Lifecycle

1. Proposal
2. Review and approval
3. Active execution
4. Periodic evaluation
5. Renewal, merge, or sunset

## Reporting

Working groups should provide monthly updates to the Core Team and SIG.
