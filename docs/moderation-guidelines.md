# Moderation Guidelines

This document defines moderation expectations for OSSAfrica spaces.

## Objectives

- Maintain a safe, respectful environment
- Enforce conduct policies consistently
- Resolve issues with clear communication

## Moderator Responsibilities

- Review reports promptly
- Apply proportional actions
- Document major moderation decisions
- Escalate high-risk incidents appropriately

## Moderation Steps

1. Review report and context.
2. Classify severity.
3. Apply response (warning, timeout, removal, escalation).
4. Record action and rationale.

## Escalation

Escalate to Core Team and SIG for severe, repeated, or sensitive cases.
