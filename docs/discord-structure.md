# Discord Structure

This document outlines recommended structure for the OSSAfrica Discord
community.

## Category Model

- Announcements
- Community
- Working Groups
- Support
- Events
- Moderation

## Baseline Channels

- `#welcome`
- `#announcements`
- `#general`
- `#introductions`
- `#help`
- `#events`
- `#resources`

## Governance Notes

- Keep channel purposes documented.
- Restrict announcement posting permissions.
- Use moderation logs for transparency where appropriate.

## Periodic Review

Review channel usage monthly and archive inactive channels.
